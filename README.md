# 📐CompassShapeFinder
2Dのカジュアルゲームです。
常に回転しながら円を描いているコンパスをクリックやタップで動かしながら、制限時間内に指定された図形を全て見つけることが目的です。

※このリポジトリは学習/ポートフォリオ公開を目的としており、一部の外部アセットはライセンス上の理由により除外しております。
